# Integration Testing

This code covers three different ways of doing integration testing on local machine or
on a CI/CD (Continuous Integration/Continuous Deployment) pipeline.

This code has three modules 

    1 Database module which uses in memory H2 DB
    2 Mockito module which uses mockito objects for storing data 
    3 TestContainers module uses testContainers in a docker environment spawning a Postgres DB.

The complete article can be found on www.rajeevkuruganti.com/IntegrationTesting
